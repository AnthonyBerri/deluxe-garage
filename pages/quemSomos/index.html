Quem Somos.

Alunos do 1º Semestre de Engenharia de Software: Ana Beatriz da Silva, Anthony Felix, Arthur Faust Locatelli, Marcos Antonio Martins, Leonardo Lamin, Wesley dos Reis Marcos.

Este projeto é fruto do trabalho conjunto de um grupo de estudantes de Engenharia de Software. O objetivo foi criar um protótipo de site para a comercialização de carros de luxo, com o uso exclusivo da ferramenta Figma, uma plataforma amplamente utilizada para o design de interfaces e protótipos interativos. Nosso desafio foi projetar uma experiência de usuário intuitiva e fluida, que fosse responsiva, se adaptando adequadamente a diferentes dispositivos e tamanhos de tela.

O principal objetivo deste projeto foi a criação de um protótipo interativo de um site de vendas de carros de luxo, com ênfase na experiência do usuário e na responsividade do design. A partir do Figma, desenvolvemos todas as telas e interações, simulando a navegação e transições entre páginas, de forma que o usuário pudesse ter uma visão clara de como seria a experiência final. Garantimos que houvesse as transições fluidas entre as telas e os demais objetos no site. Mas também que o layout fosse flexível e se ajustasse a diferentes resoluções de tela, assegurando que o design mantivesse sua estética e funcionalidade em qualquer dispositivo.

A metodologia adotada foi centrada no uso do Figma para a criação do protótipo, explorando suas funcionalidades para o desenvolvimento de interfaces de alta fidelidade. Iniciamos o projeto com o levantamento das necessidades do usuário e a definição da estrutura básica do site, incluindo o mapeamento das telas e o fluxo de navegação. A partir disso, procedemos com a construção das telas e interações, utilizando as ferramentas do Figma para garantir uma navegação fluida e a transição entre telas sem perder a consistência visual.

O público-alvo do projeto gira em torno de pessoas ricas que desejam passear sem perder seus luxos, pessoas ricas que querem experimentar em carros super sport e empresas que desejam usar os carros para fins de marketing.

Um dos principais desafios foi tornar o design responsivo, o que exigiu ajustes contínuos nas proporções e no posicionamento dos elementos para garantir que a interface se adaptasse bem ao desktop.

Este projeto tem como objetivo a criação de um protótipo interativo de um site de vendas de carros de luxo, desenvolvido exclusivamente na ferramenta Figma. O escopo do projeto abrange o design de uma interface responsiva, garantindo uma experiência de usuário intuitiva e fluida, adaptada a diferentes dispositivos e tamanhos de tela, o foco principal é a criação de um layout esteticamente agradável e funcional para a comercialização de veículos de luxo, considerando as necessidades específicas do público-alvo e as funcionalidades de navegação.
